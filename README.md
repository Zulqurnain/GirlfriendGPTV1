# GirlfriendGPTV1 - OSS AI Companion Chatbot

Build your own AI companion in Python using ChatGPT API.

## Features

- AI-powered conversation using ChatGPT
- Customizable personality and responses
- Open source and self-hostable
- Python-based, easy to extend

## Quick Start

```bash
git clone https://github.com/Zulqurnain/GirlfriendGPTV1.git
cd GirlfriendGPTV1
pip install -r requirements.txt
python main.py
```

## Requirements

- Python 3.8+
- OpenAI API key
